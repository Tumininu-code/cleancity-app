# CleanCity — AI Powered Waste Reporting App
Final Year Project · MIVA Open University · 2026
Student: Olumutimi Jesutumininu | Matric: 2023/A/SENG/0027

## Overview
CleanCity is a web-based prototype of an AI-powered mobile application for reporting waste and environmental issues in Nigerian urban areas.

## How to Run
1. Clone this repository
2. Open `index.html` in any modern browser
3. No installation required

## Tech Stack
- HTML5, CSS3, JavaScript
- Claude AI API (Anthropic) — image classification
- GitHub Pages — deployment
